# croxen-knowledge

A public knowledge base about AI, IT, automation, and personal technology.

This is a controlled projection of a private knowledge base. Content is
published only after review and explicit approval. The source of truth is
Markdown; HTML is generated for the web.

## About

This site documents practical experiments, lessons learned, and guides
related to AI tooling, home automation, IT management, and personal
technology. It is written to be useful to non-specialists — plain language,
honest about failures, no hype.

## Structure

```
content/    — approved Markdown articles, published as HTML
templates/  — Jinja2 HTML templates for the static site
static/     — CSS, images, and other static assets
build.py    — the static site generator (Markdown → HTML)
```
